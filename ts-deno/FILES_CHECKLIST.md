# Files Checklist - ts-deno/

## ‚úÖ Required Files

### 1. TypeScript Modules

- [ ] **main.ts** (Entry point)
  - Imports all modules
  - Orchestrates benchmark phases
  - Displays results
  
- [ ] **types.ts** (Type definitions)
  - BenchmarkResult interface

### 2. Configuration & Scripts

- [ ] **package.json** (Deno config)
  - Module type: "module"
  - Scripts for convenience
  
- [ ] **run.sh** (Benchmark runner)
  - Multiple iterations
  - Timing measurements
  - Result collection
  
- [ ] **generate_csv.sh** (Data generator)
  - Creates data.csv
  - 50,000 rows
  - ~5MB file

- [ ] **data.csv** (Test data)
  - Generated by generate_csv.sh
  - Not in version control
  
- [ ] **search_results.txt** (Search output)
  - Generated by file-tasks.ts
  - Not in version control

## üìù Quick Setup Commands

```bash
# 1. Create directory
mkdir -p ts-deno && cd ts-deno

# 2. Create all TypeScript files
touch main.ts types.ts cpu-tasks.ts network-tasks.ts file-tasks.ts

# 3. Create configuration
touch package.json

# 4. Create shell scripts
touch run.sh generate_csv.sh

# 5. Make scripts executable
chmod +x run.sh generate_csv.sh

# 6. Generate test data
./generate_csv.sh

# 7. Run benchmark
./run.sh
```
